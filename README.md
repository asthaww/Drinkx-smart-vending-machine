# DrinkX Smart Vending Machine

Automated protein shake vending machine powered by Raspberry Pi, built to deliver customized shakes through a touchscreen kiosk with integrated digital payments.

## Key Features

- Interactive kiosk interface for product selection
- Multiple shake sizes and add-ons
- Dynamic pricing system
- Razorpay payment integration
- Automated dispensing workflow
- GPIO-based sensor monitoring
- Motor, pump, and actuator control
- Real-time machine state handling
- Hardware testing and diagnostics modules

## Tech Stack

### Software
- Python
- Flask
- JavaScript
- HTML
- CSS

### Hardware
- Raspberry Pi
- Sensors
- Motors
- Pump system
- Dispensing mechanism

## Project Modules

- `app.py` – Main backend server and APIs  
- `controller.py` – Core vending workflow and machine control  
- `script.js` – Frontend logic and payment flow  
- `style.css` – User interface styling  
- `Testing_motors.py` – Motor diagnostics  
- `gpio_trigger.py` – GPIO trigger logic  
- `gcode_script.py` – Motion control / testing
